1.Download the zip folder on github

2.Download and install xampp server on your machine (https://www.apachefriends.org/index.html)
3.Then run  xampp and on the control panel start the following actions: apache, mysql
4.unzip the files from github in C:\xampp\htdocs\mra
5.open browser(preferablely chrome or firefox) and paste following url: http://localhost/mra/
